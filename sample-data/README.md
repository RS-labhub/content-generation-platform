# Sample Data - Reference Files Only

This folder contains **reference files only**. These files are **NOT** used by the application code.

## Purpose

These sample files serve as examples for users who want to:
- Understand how to structure persona data
- Get inspiration for creating their own personas
- Learn the format expected by the persona training system

## Files

### Persona Examples
- `persona-bap.json` - Developer Advocate style persona
- `persona-simon.json` - Tech storyteller style persona  
- `persona-rohan-sharma.json` - Technical problem solver style persona

### Content Examples
- `rohan-sharma-posts.txt` - Example LinkedIn/social media posts
- `rohan-sharma-blogs.txt` - Example blog content

## How to Use

1. **View these files** to understand the persona structure
2. **Create your own persona** through the UI's Persona Training dialog
3. **Upload your own content** to train the AI on your writing style

## Important Notes

- These files are NOT automatically loaded by the application
- Users must create their own personas through the UI
- The AI only uses persona data when explicitly selected by the user
- All user-created personas are stored in browser localStorage
